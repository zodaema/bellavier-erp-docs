## 📱 **ระบบที่ใช้: PWA Scan Station**

### **วิธีเข้าใช้:**
1. เปิดเว็บบราวเซอร์ (Chrome/Safari)
2. พิมพ์: `http://bellavier-erp/?p=pwa_scan`
3. กด [Add to Home Screen] (ครั้งแรกเท่านั้น)
4. เปิดจากหน้าจอโทรศัพท์ได้เลย!

---

## 🎯 **การใช้งานพื้นฐาน (3 ขั้นตอน)**

### **ขั้นที่ 1: สแกน QR Code**
```
📷 กด [Scan QR]
   ↓
   สแกน QR บน:
   - ใบงาน (Job Ticket)
   - Serial number
   - ชิ้นงาน
   ↓
   แสดงข้อมูลงาน
```

### **ขั้นที่ 2: เริ่มงาน**
```
▶️ กด [Start]
   ↓
   ระบบบันทึกเวลาเริ่มทำงาน
   ↓
   ทำงาน...
```

### **ขั้นที่ 3: เสร็จงาน**
```
✅ กด [Complete]
   ↓
   ยืนยัน "แน่ใจว่าเสร็จแล้ว?"
   ↓
   บันทึกเวลาทำงาน
   ↓
   ✅ เสร็จ!
```

## ⚠️ **สิ่งที่ต้องระวัง**

### **1. อย่ากด Complete ถ้ายังไม่เสร็จ!**
```
❌ กด Complete → ไม่สามารถ Undo ได้!
✅ ตรวจสอบก่อนกด
✅ มีหน้าต่างยืนยัน
```

### **2. จำกด Pause ก่อนหยุดเบรก!**
```
❌ ลืมกด Pause → เวลารันต่อ!
✅ กด Pause ก่อนทุกครั้ง
```

### **3. ปิดหน้าจอไม่เป็นไร!**
```
✅ ปิดหน้าจอได้
✅ เวลายังรันต่อ (ระบบนับเอง)
✅ เปิดใหม่ เวลายังถูก
```

```
11:00 - เสร็จทั้งหมด 10 ชิ้น
✅ หัวหน้าได้รับแจ้ง "สมชาย เสร็จงาน 10 ชิ้นแล้ว"
```

## 🆘 **เมื่อมีปัญหา**

### **ปัญหา: สแกน QR ไม่ขึ้น**
```
แก้ไข:
1. เช็ควินเทอร์เน็ต (ต้อง online)
2. อนุญาตกล้อง (Settings → Browser → Camera)
3. QR ชัดเจน (ไม่เบลอ)
4. ลอง Refresh หน้า
```

### **ปัญหา: กด Start แล้ว Error**
```
สาเหตุ:
- อาจมีคนอื่นทำงานนี้แล้ว
- งานถูก Complete ไปแล้ว
- ไม่มีสิทธิ์เข้าถึง

แก้ไข:
- Refresh หน้า
- เช็คกับหัวหน้า
```

### **ปัญหา: Offline ใช้งานไม่ได้**
```
เหตุผล:
- PWA ต้อง online ครั้งแรก

แก้ไข:
1. Online 1 ครั้ง
2. ระบบจะ cache ข้อมูล
3. ครั้งต่อไปใช้ offline ได้
```
